# Day 5: Feature Engineering

Features engineered:

- `Has Number`: determines if the password is digit-containing.
- `Character Diversity Score`: total lower, upper case, numeric, and special character type count.
- `Long Password`: flags passwords having 12 or more characters.
- `Estimated Score`: basic score based on password length and diversity.

Average password length: 9.42
Average character diversity score: 3.36
Average estimated score: 66.93

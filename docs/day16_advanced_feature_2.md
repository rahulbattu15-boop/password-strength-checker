# Day 16: Advanced Feature 2

## Work Completed

I implemented the second advanced feature: a security checklist and risk estimate.

## Feature Details

The checklist shows whether the password passes important rules:

- at least 12 characters
- lowercase letters
- uppercase letters
- numbers
- special characters
- not a common password
- no repeated character pattern
- no simple sequence

The risk estimate gives a simple user-friendly message such as Easy to guess, Hard to guess, or Very hard to guess.

## Result

The checker now explains password safety more clearly instead of only showing a score.

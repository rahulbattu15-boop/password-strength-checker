# Day 19: User Testing

## Work Completed

I planned and noted user testing for the Week 3 version.

## Testers

- Tester 1: student user testing weak and strong sample passwords.
- Tester 2: student user testing the generator and checklist.

## Feedback

- The score was easy to understand.
- The suggestions were useful.
- The generator made the project more helpful.
- The checklist made it clearer why a password was weak.
- The copy button needed a clear message when there was no password.

## Bugs Or Confusion Points

- Users wanted the generated password to be checked automatically.
- Users wanted the length number to update while moving the slider.
- Users wanted a simple explanation of risk, not only the score.

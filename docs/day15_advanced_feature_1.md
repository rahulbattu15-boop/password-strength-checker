# Day 15: Advanced Feature 1

## Work Completed

I implemented the first advanced feature: a password generator.

## Feature Details

The generator lets the user choose:

- password length
- lowercase letters
- uppercase letters
- numbers
- special characters

The generated password is added directly into the checker input so the user can immediately test it.

## Result

The project can now help users create a stronger sample password, not only check an existing one.

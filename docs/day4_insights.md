# Day 4: Data Cleaning

Day 4 is about cleaning the data for the password strength checker.

The original data had 10000 rows and 6 columns.

After cleaning it the data still has 10000 rows and 6 columns.

No duplicate rows were found in the password strength checker data.

There are no missing values in the password strength checker data after it was cleaned.

The following steps were done to clean the password strength checker data:

* Removed duplicate rows from the password strength checker data.

* Filled in the missing password values with text in the password strength checker data.

* Recalculated the length of each password from the password text in the password strength checker data.

* Recalculated the flags, for uppercase and special characters in the password strength checker data.

* Made the strength label text the same in the password strength checker data.

The cleaned password strength checker data is saved at "D:\password-strength-checker\data\cleaned_passwords_dataset.csv"

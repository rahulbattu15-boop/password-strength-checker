# Demo Video Script

Target length: 3 to 5 minutes.

## 1. Intro

Hello, my name is Rahul Battu. My CodeZoner cybersecurity project is called Password Strength Checker. This project helps users understand whether a password is weak, medium, or strong.

## 2. Live Demo

Open the live demo or `index.html`. Type a weak sample password like `rahul123` and show the Weak result with suggestions. Then type a strong sample password like `Rahul@2026#Secure` and show the Strong result.

## 3. Code Walkthrough

Show the project folders. Explain that `src/data_exploration.py` creates charts, `src/data_cleaning.py` cleans the dataset, `src/feature_engineering.py` creates new columns, and `src/password_checker.py` contains the password strength logic.

## 4. Summary

Explain that the project includes dataset analysis, charts, documentation, Python code, and a browser demo. Mention that passwords are checked locally and are not stored or uploaded.

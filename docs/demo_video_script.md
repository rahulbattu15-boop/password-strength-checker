# Demo Video Script

## 30 Second Week 2 Demo

Hello, my name is Rahul Battu. This is my Password Strength Checker project for the CodeZoner internship.

First, I will enter a weak password: `123456`. The checker marks it as Weak and gives suggestions because it is common and too simple.

Now I will enter a stronger sample password: `Rahul@2026#Secure`. The checker gives a high score and marks it as Strong because it has good length, uppercase letters, lowercase letters, numbers, and special characters.

This demo runs fully in the browser, so passwords are not saved or uploaded anywhere.

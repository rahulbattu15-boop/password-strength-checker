# Week 2 Plan

## Core Feature 1: Password Scoring

Build a scoring system that gives a password a score out of 100 based on length, lowercase letters, uppercase letters, numbers, special characters, common-password checks, repeated characters, and simple sequences.

## Core Feature 2: Password Suggestions

Give practical suggestions for weak and medium passwords. Suggestions include increasing password length, adding numbers, adding symbols, avoiding common passwords, and avoiding simple patterns.

## Core Feature 3: Browser Demo

Create a simple web page where users can enter a sample password and immediately see the result. The demo should not store or upload passwords.

## Testing Plan

Test the checker with:

- Empty password
- Very short password
- Common password
- Password with only letters
- Password with letters and numbers
- Strong password with letters, numbers, and symbols

# Final Submission Description

My project is called Password Strength Checker. It is a cybersecurity awareness project that helps students and general users understand whether a password is weak, medium, or strong. The project checks important password features such as length, lowercase letters, uppercase letters, numbers, special characters, common password patterns, repeated characters, and simple sequences. I used a password dataset to perform exploration, cleaning, feature engineering, and visualization. I also built a Python password-checking script and a browser-based demo where users can type a sample password and get a score out of 100 with improvement suggestions. The demo runs in the browser and does not save or upload passwords, which keeps the project safe for learning. This project is useful because many students reuse weak passwords for email, college portals, and online learning accounts. Future improvements include adding a password generator and safe leaked-password detection.
